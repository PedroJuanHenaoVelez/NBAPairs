# NBA Players Height Pair Finder

## Overview

The NBA Players Height Pair Finder is a web application that allows users to input a height in inches and discover all possible pairs of NBA players whose combined height matches the specified value. This tool leverages data retrieved from a public NBA players API and presents the results in an intuitive and user-friendly interface.

## Features

- **User Input:** Users can enter any integer value representing a height in inches.
- **Dynamic Pairing:** The application dynamically fetches data from the NBA players API and calculates all possible player pairs that match the input height.
- **Real-Time Results:** Results are displayed immediately after input, providing an efficient and seamless user experience.
- **Responsive Design:** The application is designed to work smoothly across various devices and screen sizes, ensuring accessibility and ease of use.

## Technologies Used

- **Frontend:**
  - TypeScript: Ensuring type safety and code maintainability.
  - CSS: Styling and layout management.

- **Backend:**
  - NBA Players API: Source of real-time data for all active NBA players.

- **Tools:**
  - npm: Package manager for handling dependencies.
  - Axios: For making API requests.
